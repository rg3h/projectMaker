As an example, you might create a project template with several html files that reference the name of the project. You put a placeholder {{projectTitle}} string in each of them and then use projectMaker to create new projects, telling it to replace the {{projectTitle}} with the new project name. projectMaker is simple, quick, and easy to use. Since it is written in javascript, it is easy to inspect and extend.
